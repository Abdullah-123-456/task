## Simple Form App

A static HTML/CSS/JS form that saves submitted data to an SQLite database via Node.js/Express.

### How to Run

1. Clone the repo  
   `git clone https://github.com/Abdullah-123-456/task`
2. Navigate into the folder  
   `cd task`
3. Install dependencies  
   `npm install`
4. Initialize database  
   `node init-db.js`
5. Start the server  
   `node server.js`
6. Open browser at  
   `http://localhost:3000`
